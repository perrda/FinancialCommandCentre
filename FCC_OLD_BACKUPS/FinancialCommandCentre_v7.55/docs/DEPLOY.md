# Deployment Guide

## Cloudflare Pages (Recommended)

1. Go to https://pages.cloudflare.com
2. Create new project → Direct Upload
3. Upload `financial_command_centre.html`
4. Rename to `index.html` if needed
5. Deploy

## Local Testing

Simply open `financial_command_centre.html` in any modern browser.

## PWA Installation

1. Open in Chrome/Safari
2. Click "Add to Home Screen" or install prompt
3. App works offline after first load
